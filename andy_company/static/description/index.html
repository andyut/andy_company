AU - UDF Multi Company

User Defined Fields for Multi Company

This module provides support for User Defined Fields (UDF) in a multi-company environment. It is designed for businesses that require flexible additional data across different entities.

Main Features:

  • Full support for multi-company setup
  • Easy configuration of UDF
  • Integration with standard Odoo modules
  • Licensed under LGPL-3
